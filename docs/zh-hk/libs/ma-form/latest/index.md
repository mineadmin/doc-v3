# MaForm

`@mineadmin/form` 是 MineAdmin 生態中的表單庫，負責表單配置、動態渲染、校驗、插槽和複雜表單場景。

## 當前版本

- 文檔版本：`latest`
- 包名：`@mineadmin/form`
- 版本策略：獨立發版，不跟隨 MineAdmin 主產品大版本
