# MaSearch

`@mineadmin/search` 是 MineAdmin 生態中的搜索表單庫，適合列表篩選、摺疊搜索、動態搜索項和表格聯動場景。

## 當前版本

- 文檔版本：`latest`
- 包名：`@mineadmin/search`
- 版本策略：獨立發版，不跟隨 MineAdmin 主產品大版本
