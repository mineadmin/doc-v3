# MaEcharts

A wrapper based on [echarts](https://echarts.apache.org/zh/index.html), providing a simpler usage method.

:::tip Note
If using the `@mineadmin/echarts` dependency alone, you need to register and bind the `echarts` object to `app.config.globalProperties.$echarts`. However, in `MineAdmin`, this has already been automatically registered.
:::
