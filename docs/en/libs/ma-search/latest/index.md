# MaSearch

`@mineadmin/search` is a search form library in the MineAdmin ecosystem, suitable for list filtering, collapsible searches, dynamic search items, and table linkage scenarios.

## Current Version

- Documentation version: `latest`
- Package name: `@mineadmin/search`
- Version strategy: Independently released, not following the major version of the MineAdmin main product
