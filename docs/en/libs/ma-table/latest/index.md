# MaTable

`@mineadmin/table` is a foundational table library within the MineAdmin ecosystem, providing capabilities such as sorting, filtering, pagination, multi-selection, tree tables, and custom rendering.

## Current Version

- Documentation version: `latest`
- Package name: `@mineadmin/table`
- Version strategy: Independently released, not tied to the major version of the MineAdmin main product
