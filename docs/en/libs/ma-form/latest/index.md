# MaForm

`@mineadmin/form` is the form library within the MineAdmin ecosystem, responsible for form configuration, dynamic rendering, validation, slots, and complex form scenarios.

## Current Version

- Documentation version: `latest`
- Package name: `@mineadmin/form`
- Version strategy: Independent release, not following the major version of the MineAdmin main product
