# MaProTable

`@mineadmin/pro-table` is an advanced table library within the MineAdmin ecosystem, designed for business scenarios such as data management, CRUD, search, action columns, and toolbar extensions.

## Current Version

- Document Version：`latest`
- Package Name：`@mineadmin/pro-table`
- Version Strategy：Independent release, not following the major version of the MineAdmin main product
