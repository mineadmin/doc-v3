# Independent Libraries

This section contains documentation for libraries that are independently maintained and versioned within the MineAdmin ecosystem. They do not follow the `v3` or `v4` major version release cadence of the MineAdmin main product, and are therefore placed under the `/libs/{library}/latest/` path.

## Library List

- [MaForm](/libs/ma-form/latest/) - `@mineadmin/form`, providing form rendering and dynamic form capabilities.
- [MaTable](/libs/ma-table/latest/) - `@mineadmin/table`, providing basic table capabilities.
- [MaSearch](/libs/ma-search/latest/) - `@mineadmin/search`, providing search forms and filter panels.
- [MaProTable](/libs/ma-pro-table/latest/) - `@mineadmin/pro-table`, providing advanced tables for business CRUD operations.

## Version Rules

`latest` indicates the currently recommended documentation. Fixed version paths such as `v1` or `v2` are only added when a library requires long-term retention of historical differences.
