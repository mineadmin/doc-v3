## 禁止リスト

- 宝くじ型サービス
- キャッシュバック・リベートサービス
- クラウドファンディング
- 融資・ローンサービス
- オークション
- 仮想通貨関連
- ブロックチェーン関連
- 海外決済サービス
- 個人間決済サービス
- くじ引きサービス
- プリペイドカード販売
- フォーラム型サービス
- コンテンツ収集（スクレイピング）サービス
