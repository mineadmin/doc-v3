**許可事項：**

## 4. 免責事項

### 4.1 責任制限

::: danger 重要免責条項
MineAdminのダウンロード・使用により生じたいかなる損害についても、**当チームは一切の法的責任を負いません**。
:::

#### 4.1.1 免責範囲

以下の状況を含むがこれに限らない：
- 🔸 **不測の事態** - システム障害、データ損失等
- 🔸 **過失行為** - 設定誤り、操作ミス等  
- 🔸 **契約問題** - 商業契約紛争等
- 🔸 **名誉毀損** - 誹謗、悪意ある拡散等
- 🔸 **知的財産権** - 著作権・特許権侵害等

#### 4.1.2 損害類型

全ての損害類型を免責：
- **直接損害** - 直接的な経済的損失
- **間接損害** - 利益損失、信用毀損
- **付随損害** - 派生する他の損害  
- **二次損害** - 後続的に発生する損害
