## 2. 製品の定義と許諾範囲

### 2.1 ソフトウェアの性質

**MineAdmin** は、オープンソースで無料、商用利用が可能なバックエンド権限管理システムであり、主にバックエンド管理機能を簡便に開発するために使用されます。

::: info 機能説明
**コア機能：**
- ✅ バックエンド権限管理フレームワーク
- ✅ フロントエンドテンプレートおよびコンポーネント
- ✅ 開発ツールおよびドキュメント

**機能制限：**
- ❌ MineAdmin自体は具体的な業務モジュール機能を有しません
- ❌ 業界固有のビジネスロジックは含みません
:::

### 2.2 オープンソースライセンス

MineAdminはデュアルライセンス方式を採用しています：

::: details ライセンス詳細
**Apache 2.0 ライセンス**
- エンタープライズでの使用に適しています
- 特許保護を提供します
- 著作権表示の保持が必要です

**MIT ライセンス**
- より寛容なオープンソースライセンス
- 個人開発者に適しています
- 簡素化された使用条件
:::

**お客様に可能なこと：**
- ✅ 無料での商用利用
- ✅ ソースコードの改変
- ✅ 再配布
- ✅ プライベートでの使用

## 4. 免責事項

### 4.1 責任の制限

::: danger 重要な免責条項
MineAdminをダウンロードして使用したことにより、個人または団体に生じた損害について、**本チームは一切の法的責任を負いません**。
:::

#### 4.1.1 免責の範囲

以下のような状況により生じた損害を含みますが、これらに限定されません：
- 🔸 **偶発的な事象** - システム障害、データ損失など
- 🔸 **過失行為** - 設定ミス、操作ミスなど
- 🔸 **契約上の問題** - 商業契約の紛争など
- 🔸 **風評被害** - 名誉毀損、悪意のある流布など
- 🔸 **知的財産権** - 著作権や特許権の侵害など

#### 4.1.2 損害の種類

免責はあらゆる種類の損害を対象とします：
- **直接損害** - 直接的な経済的損失
- **間接損害** - 逸失利益、のれんの毀損
- **付随的損害** - 関連して生じるその他の損害
- **派生的損害** - 結果として引き起こされる損害

### 4.2 ユーザーのリスク負担

::: warning ユーザーの責任
ユーザーは、本声明の条項に列挙されたすべての内容を明確に理解し、同意するものとし、MineAdminの使用に存在する可能性のあるリスクおよび関連する結果については、**すべてユーザー自身が負担するものとし**、本チームは一切の法的責任を負いません。
