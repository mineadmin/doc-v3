# 拡張型目次ナビゲーションデモ

このページでは、新しく実装した拡張型ページ内ナビゲーション（TOC）機能を紹介します。右側の目次ナビゲーションに注目してください。以下の特徴があります。

## 主要機能

### 視覚的階層の最適化

新しい目次ナビゲーションは、明確な視覚的階層を持ち、見出しのレベルごとにインデント、フォントサイズ、色で区別されています。

#### 見出しレベルの区別
- H1見出しは主要な章として、際立った視覚効果を持ちます
- H2-H6見出しは階層に応じて控えめに表示
- 各レベルに適切なインデントと間隔が設定されています

#### 色とスタイル
- アクティブ項目はグラデーションでハイライト表示
- ホバー効果で即時の視覚的フィードバック
- ダークモードで自動的に配色を調整

### インタラクション体験の向上

拡張されたインタラクション機能により、ナビゲーションがよりスムーズで直感的になりました。

#### スムーズスクロール
目次項目をクリックすると、突然ジャンプするのではなく、該当位置へスムーズにスクロールします。

#### スクロール同期
ページをスクロールすると、目次が自動的に現在表示中のセクションをハイライト表示します。

#### キーボードナビゲーション
キーボードショートカットによるナビゲーションをサポート：
- `↑/↓` または `j/k` - 上下移動
- `Enter` または `Space` - 現在の項目をアクティブ化
- `Home/End` - 最初/最後の項目にジャンプ
- `Esc` - モバイル版オーバーレイを閉じる

### モダンなデザイン要素

以下のようなモダンなデザイン言語を採用しています。

#### グラデーションとシャドウ
- アクティブインジケーターにエレガントなグラデーション効果
- 微妙なシャドウで奥行き感を演出
- 角丸デザインで現代的な美しさを表現

#### アニメーション遷移
- すべての状態変化にスムーズなトランジションアニメーション
- アクティブ項目のパルスアニメーションで視覚的なガイドを提供
- モバイル版のポップアップアニメーションはスムーズで自然

## 機能拡張

### 読了時間の推定

目次の上部に推定読了時間を表示し、読者の時間管理を支援します。計算には以下を考慮：
- テキストの長さ
- コードブロックの数
- 画像の数

### スクロール進捗インジケーター

ページ右側に縦方向のプログレスバーを表示し、読書の進捗を示します。

### 長い目次の折りたたみ

多数のセクションを含む長いドキュメントでは、折りたたみ/展開機能をサポート：
- Ctrl/Cmd + クリックでH1見出し以下のサブセクションを折りたたみ可能
- 折りたたみ状態を自動保存

### 検索ハイライト対応

ページ検索機能を使用すると、該当する目次項目がハイライト表示されます。

## 使用例

### 基本設定

目次ナビゲーションは自動生成され、追加設定は不要です。ただし、オプションで動作をカスタマイズ可能。

```typescript
enhanceTOC({
  enableReadingTime: true,    // 読了時間を表示
  enableProgress: true,       // 進捗バーを表示
  enableCollapse: true,       // 折りたたみ機能を有効化
  enableSmoothScroll: true,   // スムーズスクロール
  enableKeyboardNav: true,    // キーボードナビゲーション
  enableMobileFloat: true,    // モバイル版フローティング
  enableSearchHighlight: true // 検索ハイライト
})
```

### スタイルカスタマイズ

CSS変数で外観をカスタマイズ可能。

```css
:root {
  --toc-gradient-primary: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  --toc-indent-base: 1rem;
  --toc-font-h1: 0.95rem;
  /* その他の変数... */
}
```
