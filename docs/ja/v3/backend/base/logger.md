# ログ処理

## 開発モードにおけるコマンドラインログ

`.env` ファイルにおいて、`APP_DEBUG=true` の場合、サーバーは自動的にすべてのエラーログをコマンドラインに出力します。これにより開発者はローカルでのデバッグが容易になります。
`APP_DEBUG=false` の場合、サーバーは可能な限りログをデフォルトの `loggerFactor->get('xxx','default')` ログチャンネルに出力します。

::: tip

ログ使用に関する詳細なドキュメントについては、[hyperf](https://hyperf.io) のドキュメントを参照してください。本稿では基本使用法については説明しません。

:::
