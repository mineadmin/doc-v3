# 折りたたみ検索

検索フォームの折りたたみ・展開機能を表示します。検索項目が多い場合に適しており、折りたたみ機能でページスペースを節約し、ユーザー体験を向上させます。

## 折りたたみ機能の説明

### 基本設定
`fold` と `foldRows` パラメータで折りたたみ動作を制御します：

```typescript
const searchOptions = {
  fold: true,        // 折りたたみ機能を有効化
  foldRows: 2,       // デフォルトで2行を表示
}
```

### ボタンテキストのカスタマイズ
`text` 設定で折りたたみボタンのテキストをカスタマイズします：

```typescript
const searchOptions = {
  text: {
    isFoldBtn: '検索条件をさらに表示',
    notFoldBtn: '検索条件を一部折りたたむ'
  }
}
```

### プログラムによる制御
コンポーネントインスタンスのメソッドで折りたたみ状態を制御します：

```typescript
// 折りたたみ状態を切り替え
searchRef.value?.foldToggle()

// 現在の折りたたみ状態を取得
const isFold = searchRef.value?.getFold()
```

## 使用シーン

### 1. 管理システム
管理画面では複数の検索条件が必要な場合が多く、折りたたみ機能でインターフェースを整理できます。

### 2. データ分析プラットフォーム
複雑なデータフィルタ条件において、よく使う条件を表示し、高度な条件は折りたたみ領域に隠します。

### 3. モバイル対応
モバイル端末では画面スペースが限られているため、折りたたみ機能が特に重要です。

## 折りたたみ戦略

### スマート折りたたみ
- 折りたたむ検索項目数を自動計算
- 最もよく使う検索条件を優先表示
- 操作ボタンは常に表示

### レスポンシブ折りたたみ
- 画面サイズに応じて折りたたみ行数を自動調整
- モバイル端末ではデフォルトでよりコンパクトな表示
- スムーズな展開・折りたたみアニメーション

## 主な機能

- 📱 モバイル端末に優しい設計
- 🔄 スムーズな展開・折りたたみアニメーション
- 🎛 柔軟な折りたたみ行数設定
- 📝 カスタマイズ可能なボタンテキスト
- ⚡ プログラムによる状態制御

## ベストプラクティス

### 1. 適切な折りたたみ行数設定
実際の業務シーンとページレイアウトに応じて、`foldRows` パラメータを適切に設定します：

```typescript
// デスクトップ推奨 2-3 行
// モバイル推奨 1-2 行
const searchOptions = {
  foldRows: window.innerWidth < 768 ? 1 : 2
}
```

### 2. 検索項目の順序最適化
よく使う検索条件を前に配置し、折りたたみ状態でも表示されるようにします。

### 3. 状態表示の提供
カスタムボタンテキストやアイコンで、現在の折りたたみ状態をユーザーに明確に示します。
