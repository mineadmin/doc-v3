# プラグイン単体テスト

標準アプリケーションディレクトリ構成の説明

---

## 本質的にプラグインもcomposerパッケージですが、mineはこれを基盤にさらに多くの機能を統合しています。

単体テストを実施したい開発者は、プラグインディレクトリ直下にcomposer.jsonファイルを作成してください。

ただし、以下の点に注意する必要があります:

1. composer.jsonとmine.jsonのpsr4名前空間が一致しているか確認してください
2. 注意事項: [公式以外のプラグインのインストールは禁止です](#){style="color: red;"}。アプリケーションが悪意のある破壊を受ける可能性があります
