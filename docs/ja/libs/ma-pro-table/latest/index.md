# MaProTable

`@mineadmin/pro-table` は、MineAdminエコシステムにおける高度なテーブルライブラリであり、データ管理、CRUD、検索、操作列、ツールバー拡張などのビジネスシナリオに対応しています。

## 現在のバージョン

- ドキュメントバージョン：`latest`
- パッケージ名：`@mineadmin/pro-table`
- バージョン戦略：独立したリリース、MineAdminメイン製品のメジャーバージョンに依存しない
