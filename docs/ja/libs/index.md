# 独立ライブラリ

ここには、MineAdmin エコシステム内で独立してメンテナンスされ、独立してリリースされるライブラリのドキュメントを収録しています。これらは MineAdmin メイン製品の `v3`、`v4` メジャーバージョンのリズムに従わないため、一律 `/libs/{library}/latest/` パスに配置されます。

## ライブラリ一覧

- [MaForm](/libs/ma-form/latest/) - `@mineadmin/form`、フォームレンダリングと動的フォーム機能。
- [MaTable](/libs/ma-table/latest/) - `@mineadmin/table`、基本テーブル機能。
- [MaSearch](/libs/ma-search/latest/) - `@mineadmin/search`、検索フォームとフィルターパネル。
- [MaProTable](/libs/ma-pro-table/latest/) - `@mineadmin/pro-table`、ビジネス CRUD 向けの高度なテーブル。

## バージョンルール

`latest` は現在推奨されるドキュメントを示します。特定のライブラリで過去の差分を長期間保持する必要がある場合にのみ、`v1`、`v2` などの固定バージョンパスが追加されます。
