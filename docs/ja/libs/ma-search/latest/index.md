# MaSearch

`@mineadmin/search` は MineAdmin エコシステムの検索フォームライブラリであり、リストフィルタリング、折りたたみ検索、動的検索項目、およびテーブル連携のシナリオに適しています。

## 現在のバージョン

- ドキュメントバージョン：`latest`
- パッケージ名：`@mineadmin/search`
- バージョン戦略：独立したリリース、MineAdmin メイン製品のメジャーバージョンに追従しません
