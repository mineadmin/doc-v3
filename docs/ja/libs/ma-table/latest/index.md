# MaTable

`@mineadmin/table` は、MineAdmin エコシステムの基本テーブルライブラリであり、並び替え、フィルタリング、ページネーション、複数選択、ツリーテーブル、カスタムレンダリングなどの機能を提供します。

## 現在のバージョン

- ドキュメントバージョン：`latest`
- パッケージ名：`@mineadmin/table`
- バージョン戦略：独立してリリースされ、MineAdmin メイン製品のメジャーバージョンには追随しません。
