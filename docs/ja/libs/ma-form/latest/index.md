# MaForm

`@mineadmin/form`は、MineAdminエコシステムのフォームライブラリであり、フォーム設定、動的レンダリング、バリデーション、スロット、複雑なフォームシナリオを担当します。

## 現在のバージョン

- ドキュメントバージョン：`latest`
- パッケージ名：`@mineadmin/form`
- バージョン戦略：独立したリリース、MineAdminメイン製品のメジャーバージョンには従わない
