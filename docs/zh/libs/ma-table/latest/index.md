# MaTable

`@mineadmin/table` 是 MineAdmin 生态中的基础表格库，提供排序、筛选、分页、多选、树形表格和自定义渲染等能力。

## 当前版本

- 文档版本：`latest`
- 包名：`@mineadmin/table`
- 版本策略：独立发版，不跟随 MineAdmin 主产品大版本
