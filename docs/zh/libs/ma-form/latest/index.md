# MaForm

`@mineadmin/form` 是 MineAdmin 生态中的表单库，负责表单配置、动态渲染、校验、插槽和复杂表单场景。

## 当前版本

- 文档版本：`latest`
- 包名：`@mineadmin/form`
- 版本策略：独立发版，不跟随 MineAdmin 主产品大版本
