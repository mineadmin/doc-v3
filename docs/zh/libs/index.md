# 独立库

这里收录 MineAdmin 生态中独立维护、独立发版的库文档。它们不跟随 MineAdmin 主产品的 `v3`、`v4` 大版本节奏，因此统一放在 `/libs/{library}/latest/` 路径下。

## 库列表

- [MaForm](/libs/ma-form/latest/) - `@mineadmin/form`，表单渲染与动态表单能力。
- [MaTable](/libs/ma-table/latest/) - `@mineadmin/table`，基础表格能力。
- [MaSearch](/libs/ma-search/latest/) - `@mineadmin/search`，搜索表单与筛选面板。
- [MaProTable](/libs/ma-pro-table/latest/) - `@mineadmin/pro-table`，面向业务 CRUD 的高级表格。

## 版本规则

`latest` 表示当前推荐文档。只有某个库需要长期保留历史差异时，才会新增 `v1`、`v2` 等固定版本路径。
