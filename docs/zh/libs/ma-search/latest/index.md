# MaSearch

`@mineadmin/search` 是 MineAdmin 生态中的搜索表单库，适合列表筛选、折叠搜索、动态搜索项和表格联动场景。

## 当前版本

- 文档版本：`latest`
- 包名：`@mineadmin/search`
- 版本策略：独立发版，不跟随 MineAdmin 主产品大版本
