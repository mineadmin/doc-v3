# MaProTable

`@mineadmin/pro-table` 是 MineAdmin 生態中的高階表格庫，面向資料管理、CRUD、搜尋、操作列和工具欄擴充套件等業務場景。

## 當前版本

- 文件版本：`latest`
- 包名：`@mineadmin/pro-table`
- 版本策略：獨立發版，不跟隨 MineAdmin 主產品大版本
