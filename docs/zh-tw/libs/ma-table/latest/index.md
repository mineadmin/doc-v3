# MaTable

`@mineadmin/table` 是 MineAdmin 生態中的基礎表格庫，提供排序、篩選、分頁、多選、樹形表格和自定義渲染等能力。

## 當前版本

- 文件版本：`latest`
- 包名：`@mineadmin/table`
- 版本策略：獨立發版，不跟隨 MineAdmin 主產品大版本
