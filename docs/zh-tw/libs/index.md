# 獨立庫

這裡收錄 MineAdmin 生態中獨立維護、獨立發版的庫文件。它們不跟隨 MineAdmin 主產品的 `v3`、`v4` 大版本節奏，因此統一放在 `/libs/{library}/latest/` 路徑下。

## 庫列表

- [MaForm](/libs/ma-form/latest/) - `@mineadmin/form`，表單渲染與動態表單能力。
- [MaTable](/libs/ma-table/latest/) - `@mineadmin/table`，基礎表格能力。
- [MaSearch](/libs/ma-search/latest/) - `@mineadmin/search`，搜尋表單與篩選面板。
- [MaProTable](/libs/ma-pro-table/latest/) - `@mineadmin/pro-table`，面向業務 CRUD 的高階表格。

## 版本規則

`latest` 表示當前推薦文件。只有某個庫需要長期保留歷史差異時，才會新增 `v1`、`v2` 等固定版本路徑。
